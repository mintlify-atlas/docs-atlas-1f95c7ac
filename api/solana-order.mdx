## Overview

The `SvmCrossChainOrder` class represents a cross-chain swap order originating from Solana. It extends `BaseOrder` and encapsulates all order parameters, escrow details, and auction mechanics specific to Solana.

## Class Definition

```typescript
class SvmCrossChainOrder extends BaseOrder<SolanaAddress, SolanaOrderJSON>
```

## Constructor

```typescript
new SvmCrossChainOrder(
  orderInfo: OrderInfoData,
  escrowParams: SolanaEscrowParams,
  details: SolanaDetails,
  extra: SolanaExtra
)
```

<Warning>
  The constructor is private. Use static factory methods `new()`, `fromContractOrder()`, or `fromJSON()` to create instances.
</Warning>

#### Example

```typescript
import { 
  SvmCrossChainOrder, 
  SolanaAddress, 
  EvmAddress,
  HashLock,
  TimeLocks,
  AuctionDetails,
  NetworkEnum
} from '@1inch/cross-chain-sdk'

const order = SvmCrossChainOrder.new(
  {
    srcToken: SolanaAddress.NATIVE, // SOL
    dstToken: EvmAddress.fromString('0x...'), // USDC on Ethereum
    maker: SolanaAddress.fromString('maker...'),
    srcAmount: 1000000000n, // 1 SOL (9 decimals)
    minDstAmount: 150000000n, // 150 USDC (6 decimals)
    receiver: EvmAddress.fromString('0x...')
  },
  {
    hashLock: HashLock.forSingleFill(secret),
    srcChainId: NetworkEnum.SOLANA,
    dstChainId: 1, // Ethereum
    srcSafetyDeposit: 1000000n,
    dstSafetyDeposit: 1000000n,
    timeLocks: TimeLocks.new(...)
  },
  {
    auction: AuctionDetails.new(...)
  },
  {
    allowMultipleFills: false
  }
)
```

## Resolver Cancellation Config

The `ResolverCancellationConfig` controls whether and how resolvers can cancel orders.

```typescript
class ResolverCancellationConfig {
  constructor(
    public readonly maxCancellationPremium: bigint,  // u64
    public readonly cancellationAuctionDuration: number
  )
}
```

### Predefined Configs

<ResponseField name="ResolverCancellationConfig.ZERO" type="ResolverCancellationConfig">
  Disables resolver cancellation (both values are 0)
</ResponseField>

<ResponseField name="ResolverCancellationConfig.ALMOST_ZERO" type="ResolverCancellationConfig">
  Enables resolver cancellation with minimal premium (values are 1)
</ResponseField>

### Methods

## Example: Complete Order Flow

```typescript
import { 
  SvmCrossChainOrder,
  SolanaAddress,
  EvmAddress,
  HashLock,
  TimeLocks,
  AuctionDetails,
  NetworkEnum
} from '@1inch/cross-chain-sdk'

// 1. Create order
const secret = crypto.randomBytes(32)
const hashLock = HashLock.forSingleFill(secret)

const order = SvmCrossChainOrder.new(
  {
    srcToken: SolanaAddress.fromString('So11111111111111111111111111111111111111112'), // Wrapped SOL
    dstToken: EvmAddress.fromString('0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48'), // USDC on Ethereum
    maker: makerAddress,
    srcAmount: 2_000_000_000n, // 2 SOL
    minDstAmount: 300_000_000n, // 300 USDC
    receiver: receiverAddress
  },
  {
    hashLock,
    srcChainId: NetworkEnum.SOLANA,
    dstChainId: 1,
    srcSafetyDeposit: 10_000_000n,
    dstSafetyDeposit: 1_000_000n,
    timeLocks: TimeLocks.new(
      3600n,  // publicDeadline: 1 hour
      7200n,  // forcedExit: 2 hours  
      10800n, // safetyDeposit: 3 hours
      14400n  // cancellation: 4 hours
    )
  },
  {
    auction: AuctionDetails.new({
      startTime: BigInt(Math.floor(Date.now() / 1000)),
      duration: 180n, // 3 minutes
      initialRateBump: 100000, // 10%
      points: [
        { delay: 60, coefficient: 50000 },
        { delay: 120, coefficient: 25000 },
        { delay: 180, coefficient: 0 }
      ]
    })
  },
  {}
)

// 2. Get order hash
const orderHash = order.getOrderHash(NetworkEnum.SOLANA)
console.log('Order hash:', orderHash)

// 3. Get order account (PDA)
const orderAccount = order.getOrderAccount(escrowProgramId)

// 4. Submit order to API and on-chain
const orderJSON = order.toJSON()
await sdk.submitOrder(orderJSON, signature)

// 5. After fill, get escrow addresses
const escrowAddress = order.getSrcEscrowAddress(
  escrowProgramId,
  takerAddress
)

const escrowATA = order.getSrcEscrowATA({
  programId: escrowProgramId,
  taker: takerAddress,
  tokenProgramId: TOKEN_PROGRAM_ID
})
```
