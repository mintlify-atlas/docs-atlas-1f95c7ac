title: "Quote"
description: "Quote class for cross-chain swap quotes and order creation"
---

The `Quote` class represents a cross-chain swap quote returned from the API. It contains pricing information, auction parameters, and methods to create orders.

## Type Parameters

```typescript
class Quote<
  SrcChain extends SupportedChain = SupportedChain,
  DstChain extends SupportedChain = SupportedChain
>
```

<ParamField path="SrcChain" type="SupportedChain">
  Source chain type (EVM or Solana)
</ParamField>

<ParamField path="DstChain" type="SupportedChain">
  Destination chain type (EVM or Solana)
</ParamField>

## Instance Methods

### createEvmOrder()

Creates an EVM cross-chain order from the quote.

```typescript
const order = quote.createEvmOrder({
  hashLock: HashLock.forSingleFill(secret),
  receiver: EvmAddress.fromString('0x...'),
  preset: PresetEnum.fast,
  nonce: 123456n
})
```

<Warning>
  Can only be called on quotes with EVM source chain. Use `isEvmQuote()` to check.
</Warning>

#### Parameters

<ParamField path="params" type="EvmCrossChainOrderParamsData" required>
  <ParamField path="hashLock" type="HashLock" required>
    Hash lock for escrow security
  </ParamField>
  
  <ParamField path="receiver" type="AddressForChain<SupportedChain>">
    Receiver address on destination chain (defaults to maker)
  </ParamField>
  
  <ParamField path="preset" type="PresetEnum">
    Auction preset (defaults to recommended preset)
  </ParamField>
  
  <ParamField path="nonce" type="bigint">
    Order nonce (auto-generated if not provided for required cases)
  </ParamField>
  
  <ParamField path="permit" type="string">
    Permit signature for gasless approval
  </ParamField>
  
  <ParamField path="isPermit2" type="boolean">
    Use Permit2 for token approval
  </ParamField>
  
  <ParamField path="delayAuctionStartTimeBy" type="bigint">
    Delay auction start time by this many seconds
  </ParamField>
  
  <ParamField path="orderExpirationDelay" type="bigint">
    Order expiration delay after auction ends (default: 12s)
  </ParamField>
</ParamField>

#### Returns

<ResponseField name="EvmCrossChainOrder" type="EvmCrossChainOrder">
  EVM cross-chain order instance
</ResponseField>

### isEvmQuote()

Type guard to check if this is an EVM quote.

```typescript
if (quote.isEvmQuote()) {
  // TypeScript knows this is Quote<EvmChain>
  const order = quote.createEvmOrder(params)
}
```

#### Returns

<ResponseField name="boolean" type="boolean">
  `true` if source chain is EVM, `false` otherwise
</ResponseField>

### isSolanaQuote()

Type guard to check if this is a Solana quote.

```typescript
if (quote.isSolanaQuote()) {
  // TypeScript knows this is Quote<SolanaChain>

```

#### Returns

<ResponseField name="boolean" type="boolean">
  `true` if source chain is Solana, `false` otherwise

## Usage Examples

### Basic Quote to Order Flow

```typescript
import { SDK, HashLock, PresetEnum } from '@1inch/cross-chain-sdk'
import { randomBytes } from 'crypto'

// Get quote
const quote = await sdk.getQuote({
  srcChainId: 1,
  dstChainId: 56,
  srcTokenAddress: '0x...',
  dstTokenAddress: '0x...',
  amount: '1000000000000000000',
  walletAddress: '0x...',
  enableEstimate: true
})

// Create hash lock
const secret = '0x' + randomBytes(32).toString('hex')
const secretHash = HashLock.hashSecret(secret)
const hashLock = HashLock.forSingleFill(secret)

// Create order
if (quote.isEvmQuote()) {
  const order = quote.createEvmOrder({
    hashLock,
    receiver: EvmAddress.fromString('0x...'),
    preset: PresetEnum.fast
  })
  
  console.log('Order created:', order.getOrderHash(quote.srcChainId))
}
```

### Multiple Fills Order

```typescript
import { randomBytes } from 'crypto'

// Generate secrets for multiple fills
const secrets = Array.from({ length: 3 }, () => 
  '0x' + randomBytes(32).toString('hex')
)

const secretHashes = secrets.map(HashLock.hashSecret)
const leaves = HashLock.getMerkleLeavesFromSecretHashes(secretHashes)
const hashLock = HashLock.forMultipleFills(leaves)

// Create order allowing multiple fills
const order = quote.createEvmOrder({
  hashLock,
  receiver: EvmAddress.fromString('0x...'),
  preset: PresetEnum.medium
})
```

### Custom Preset

```typescript
// Get quote with custom auction parameters
const quote = await sdk.getQuoteWithCustomPreset(
  {
    srcChainId: 1,
    dstChainId: 56,
    srcTokenAddress: '0x...',
    dstTokenAddress: '0x...',
    amount: '1000000000000000000',
    walletAddress: '0x...',
    enableEstimate: true
  },
  {
    customPreset: {
      auctionDuration: 300,
      auctionStartAmount: '990000000000000000',
      auctionEndAmount: '950000000000000000',
      initialRateBump: 10000,
      points: [
        { delay: 30, coefficient: 9900 },
        { delay: 120, coefficient: 9700 },
        { delay: 240, coefficient: 9500 }
      ]
    }
  }
)

// Use custom preset
const order = quote.createEvmOrder({
  hashLock,
  preset: PresetEnum.custom
})
```

### Checking Quote Properties

```typescript
console.log('Source amount:', quote.srcTokenAmount)
console.log('Destination amount:', quote.dstTokenAmount)
console.log('Recommended preset:', quote.recommendedPreset)
console.log('Slippage:', quote.slippage)

// Get preset details
const preset = quote.getPreset()
console.log('Auction duration:', preset.auctionDuration)
console.log('Start amount:', preset.auctionStartAmount)
console.log('End amount:', preset.auctionEndAmount)
```
