## Overview

The `EvmCrossChainOrder` class represents a cross-chain swap order on EVM-compatible blockchains (Ethereum, Polygon, Arbitrum, etc.). It extends `BaseOrder` and encapsulates all order parameters, escrow details, and auction mechanics.

## Class Definition

```typescript
class EvmCrossChainOrder extends BaseOrder<EvmAddress, LimitOrderV4Struct>
```

## Constructor

```typescript
new EvmCrossChainOrder(
  extension: EscrowExtension,
  orderInfo: OrderInfoData,
  extra?: EvmExtra
)
```

<Warning>
  The constructor is protected. Use static factory methods `new()`, `fromNative()`, or `fromDataAndExtension()` to create instances.
</Warning>

```typescript

  escrowFactory,
  {
    makerAsset: EvmAddress.fromString('0x...'), // USDC on Ethereum
    takerAsset: EvmAddress.fromString('0x...'), // USDC on Polygon
    makingAmount: 1000000000n, // 1000 USDC
    takingAmount: 999000000n,  // 999 USDC minimum
    maker: EvmAddress.fromString('0x...')
  },
  {
    hashLock: HashLock.forSingleFill(secret),
    srcChainId: 1, // Ethereum
    dstChainId: 137, // Polygon
    srcSafetyDeposit: 10000000n,
    dstSafetyDeposit: 10000000n,
    timeLocks: TimeLocks.new(...)
  },
  {
    auction: AuctionDetails.new(...),
    whitelist: []
  }
)
```

### fromNative()

Create an order from native asset (ETH, MATIC, etc.).

```typescript
EvmCrossChainOrder.fromNative(
  chainId: number,
  ethOrdersFactory: ProxyFactory,
  escrowFactory: EvmAddress,
  orderInfo: Omit<EvmCrossChainOrderInfo, 'makerAsset'>,
  details: EvmDetails,
  escrowParams: EvmEscrowParams,
  extra?: EvmExtra
): EvmCrossChainOrder
```

<Note>
  Native orders must be submitted on-chain through `NativeOrderFactory.create` AND off-chain to the relayer.
</Note>

<ParamField path="chainId" type="number" required>
  Source chain ID
</ParamField>

<ParamField path="ethOrdersFactory" type="ProxyFactory" required>
  Native order factory contract instance
</ParamField>

<ParamField path="escrowFactory" type="EvmAddress" required>
  Escrow factory contract address
</ParamField>

#### Example

```typescript
// Create order swapping ETH on Ethereum to USDC on Polygon
const order = EvmCrossChainOrder.fromNative(
  1, // Ethereum
  ethOrderFactory,
  escrowFactory,
  {
    takerAsset: usdcPolygon,
    makingAmount: parseEther('1'), // 1 ETH
    takingAmount: 2000000000n, // 2000 USDC
    maker: userAddress
  },
  details,
  escrowParams
)
```
